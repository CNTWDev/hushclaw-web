API Access

Use the API endpoints below to obtain a token programmatically, then pass it as a Bearer token in every request.

# Step 1 — request a code

curl -X POST {{ request.base_url }}api/v1/auth/send-code \

-H 'Content-Type: application/json' \

-d '{"email": "you@example.com"}'

# Step 2 — exchange code for token

curl -X POST {{ request.base_url }}api/v1/auth/verify-code \

-H 'Content-Type: application/json' \

-d '{"email": "you@example.com", "code": "123456", "client_name": "My Script"}'

# Step 3 — use the token

curl {{ request.base_url }}api/v1/account/me \

-H 'Authorization: Bearer hc_...'
